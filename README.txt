Afin de lancer le programme, ouvrer dans un terminal le fichier Main.py

Les dossier "Nuages" comportant l'ensembles des nuages de points et 
automatiquement créer avec des nuages.

L'ensemble des scripts est rédigé en UTF-8, la docstring et les commentaires
respectent la norme PEP 484 (https://www.python.org/dev/peps/pep-0484/) ainsi
sue la norme Google (http://google.github.io/styleguide/pyguide.html).

Le programme test.py ne fonctionne que si la bibliothèque openpyxl est instalée.
(https://openpyxl.readthedocs.io/en/stable/)

Le fichier Data.xlsx comporte les données brutes. Elles sont traité automatiquement dans Performance.xlsx
